# 🎫 Helpdesk & ITSM Ticketing Simulations

Welcome to my Helpdesk and IT Service Management (ITSM) portfolio. 

As part of my transition into Enterprise IT, I use this directory to document simulated IT Helpdesk tickets. The goal of these case studies is to demonstrate my systematic approach to troubleshooting, incident management, and customer communication.

## Key Competencies Demonstrated
* **ITSM Familiarization:** Logging, updating, and resolving tickets using industry-standard formats (simulating Jira Service Management).
* **Remote Troubleshooting:** Diagnosing and remediating OS, hardware, and driver conflicts remotely.
* **Customer Communication:** Utilizing the "Sandwich Method" to deliver technical updates to end-users with empathy and clarity.
* **Enterprise Awareness:** Bridging the gap between residential/BYOD break-fix solutions and scalable, centrally managed enterprise deployments (e.g., Microsoft Intune, SCCM).
